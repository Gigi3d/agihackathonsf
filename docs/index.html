How it works

One webhook. Two agents. Four platforms.

A GitHub event fires a Runtype flow that dedupes, runs a two-agent swarm, finds a meme, and fans out, with per-platform rendering so the gif actually plays.

🧠

Grounded, not hype

The Copywriter writes only from the Tech Lead's brief, no invented facts, numbers, or features.

🔁

Unattended X posting

A self-refreshing OAuth2 flow rotates its own token, so tweets keep posting without a human.

✅

Eval-gated reliability

A Runtype eval suite grades every announcement, it caught a real model regression before it shipped.

Multi-agent coordination

A swarm, not a wrapper.

Work is delegated across two specialized agents with a visible hand-off. The same coordination runs on a COTAL mesh with signed identity, roles, and a durable message log.
